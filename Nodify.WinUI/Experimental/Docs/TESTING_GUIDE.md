# Node Editor Testing Guide

本指南介绍如何测试节点编辑器的各项功能。

## 测试环境

### 启动示例页面
1. 运行项目
2. 导航到 `SamplePage`
3. 你会看到三个预设的示例节点

## 功能测试清单

### ✅ 1. 节点创建
- **方法 1**: 在空白画布区域双击（不在节点上）
- **方法 2**: 点击工具栏的 "Add Node" 按钮
- **快捷键**: `Ctrl + N`
- **期望结果**: 在鼠标位置创建一个新节点

### ✅ 2. 节点选择
- **操作**: 单击任意节点
- **期望结果**: 
  - 节点高亮显示（视觉反馈）
  - 右侧属性面板显示节点详情
  - "Delete Selected" 按钮变为可用

### ✅ 3. 节点编辑（通过属性面板）

#### 编辑标题
1. 选择一个节点
2. 在右侧属性面板的 "Title" 文本框中修改标题
3. 按 Enter 或点击其他位置确认
4. **期望结果**: 节点标题立即更新

#### 编辑内容
1. 选择一个节点
2. 在属性面板的 "Content" 文本框中修改内容
3. **期望结果**: 节点内容立即更新

### ✅ 4. 端口管理

#### 添加输入端口
1. 选择一个节点
2. 在属性面板的 "Input Ports" 区域点击 "Add Input" 按钮
3. **期望结果**: 添加一个新的输入端口（默认名称 "Input X"）

#### 添加输出端口
1. 选择一个节点
2. 在属性面板的 "Output Ports" 区域点击 "Add Output" 按钮
3. **期望结果**: 添加一个新的输出端口（默认名称 "Output X"）

#### 编辑端口名称
1. 选择一个节点
2. 在属性面板的端口列表中，找到要编辑的端口
3. 点击端口名称旁的 "Rename" 按钮
4. 在弹出的对话框中输入新名称
5. 点击 "OK"
6. **期望结果**: 端口名称更新

#### 删除端口
1. 选择一个节点
2. 在属性面板的端口列表中，找到要删除的端口
3. 点击端口旁的 "Delete" 按钮
4. **期望结果**: 端口被删除，相关连接也被删除

### ✅ 5. 节点移动
- **操作**: 拖动节点
- **期望结果**: 
  - 节点平滑移动
  - 连接线跟随更新
  - 属性面板中的位置坐标实时更新

### ✅ 6. 节点删除

#### 方法 1: 使用按钮
1. 选择一个节点
2. 点击工具栏的 "Delete Selected" 按钮
3. **期望结果**: 节点被删除，相关连接也被删除

#### 方法 2: 使用快捷键
1. 选择一个节点
2. 按 `Delete` 键
3. **期望结果**: 节点被删除

### ✅ 7. 连接创建
1. 从一个节点的输出端口拖动
2. 拖到另一个节点的输入端口
3. 释放鼠标
4. **期望结果**: 创建一条连接线

### ✅ 8. 连接删除
1. 选择一个有连接的节点
2. 在属性面板的 "Connections" 区域找到连接
3. 点击连接旁的 "Delete" 按钮
4. **期望结果**: 连接被删除

### ✅ 9. 画布操作

#### 平移
- **操作**: 在空白区域拖动鼠标（不双击）
- **期望结果**: 画布平移

#### 缩放
- **方法 1**: 使用鼠标滚轮
- **方法 2**: 点击工具栏的 "Reset Zoom" 按钮
- **快捷键**: `Ctrl + 0` 重置缩放
- **期望结果**: 
  - 画布缩放
  - 工具栏显示当前缩放比例

## 工具栏说明

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| Add Node | Ctrl+N | 在画布中心添加新节点 |
| Delete Selected | Delete | 删除当前选中的节点 |
| Reset Zoom | Ctrl+0 | 重置缩放为 100% |

## 属性面板说明

### 节点未选中时
显示提示: "Select a node to view properties"

### 节点选中时
- **基本信息**
  - ID: 节点的唯一标识符（只读）
  - Title: 节点标题（可编辑）
  - Content: 节点内容（可编辑）
  
- **位置信息**
  - X: 横坐标（只读，拖动节点更新）
  - Y: 纵坐标（只读，拖动节点更新）

- **端口管理**
  - Input Ports: 输入端口列表
    - 显示端口名称
    - Rename 按钮：重命名端口
    - Delete 按钮：删除端口
  - Output Ports: 输出端口列表
    - 显示端口名称
    - Rename 按钮：重命名端口
    - Delete 按钮：删除端口

- **连接信息**
  - 显示所有与该节点相关的连接
  - Delete 按钮：删除连接

## 已知问题和限制

### 双击创建节点
- ✅ **已修复**: 现在只在空白区域双击才会创建节点
- 在节点上双击不会创建新节点

### 节点编辑
- ✅ **已实现**: 通过属性面板编辑标题和内容
- 未来可能支持直接在节点上双击编辑

### 端口编辑
- ✅ **已实现**: 通过属性面板管理端口
- 支持添加、删除、重命名端口
- 删除端口时会自动删除相关连接

## 测试场景示例

### 场景 1: 创建一个简单的工作流
1. 双击空白区域创建节点 A
2. 双击另一个位置创建节点 B
3. 选择节点 A，在属性面板修改标题为 "开始"
4. 选择节点 B，修改标题为 "结束"
5. 从节点 A 的输出端口拖到节点 B 的输入端口
6. **期望**: 创建了一个简单的两节点工作流

### 场景 2: 自定义节点端口
1. 创建一个新节点
2. 选择该节点
3. 在属性面板添加 3 个输入端口
4. 重命名第一个输入端口为 "参数1"
5. 重命名第二个输入端口为 "参数2"
6. 删除第三个输入端口
7. **期望**: 节点有两个自定义命名的输入端口

### 场景 3: 删除节点及其连接
1. 创建三个节点 A、B、C
2. 连接 A → B → C
3. 选择节点 B
4. 按 Delete 键
5. **期望**: 节点 B 被删除，A 和 C 之间的连接也被删除

## 性能指标

- **连接创建**: 拖动时应该有平滑的预览线
- **属性更新**: 应该实时反映到 UI
