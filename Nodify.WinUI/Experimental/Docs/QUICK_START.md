# Quick Start Guide - Node Editor

欢迎使用 Nodify.WinUI 节点编辑器！本指南将帮助你快速上手。

## 🚀 启动

1. 运行项目
2. 导航到 `SamplePage`
3. 你会看到三个预设的示例节点：
   - **Start** (左侧): 1 个输入，2 个输出
   - **Process** (中间): 2 个输入，2 个输出
   - **Output** (右侧): 2 个输入，1 个输出

## 📝 基本操作

### 创建节点
有三种方式创建新节点：
1. **双击** 空白画布区域
2. 点击工具栏的 **"Add Node"** 按钮
3. 按快捷键 **Ctrl+N**

### 选择节点
- 单击任意节点即可选中
- 选中后节点会高亮显示
- 右侧属性面板会显示节点详情

### 移动节点
- 直接拖动节点即可移动
- 连接线会自动跟随

### 删除节点
1. 先选中节点
2. 然后：
   - 点击工具栏的 **"Delete Selected"** 按钮，或
   - 按 **Delete** 键

## ✏️ 编辑节点

### 修改标题和内容
1. 选中一个节点
2. 在右侧属性面板中：
   - 修改 **Title** 文本框来更改标题
   - 修改 **Content** 文本框来更改内容
3. 更改会立即反映到节点上

## 🔌 管理端口

### 添加端口
1. 选中节点
2. 在属性面板中：
   - 点击 **"Add Input"** 添加输入端口
   - 点击 **"Add Output"** 添加输出端口

### 重命名端口
1. 选中节点
2. 在属性面板的端口列表中：
   - 找到要重命名的端口
   - 点击 **"Rename"** 按钮
   - 输入新名称并点击 **OK**

### 删除端口
1. 选中节点
2. 在属性面板的端口列表中：
   - 找到要删除的端口
   - 点击 **"Delete"** 按钮
   - 相关的连接会自动删除

## 🔗 创建连接

1. 将鼠标移到一个节点的 **输出端口**（圆点）
2. 按住鼠标左键并拖动
3. 拖到另一个节点的 **输入端口**
4. 释放鼠标
5. 连接创建成功！

### 删除连接
1. 选中连接所在的节点
2. 在属性面板的 **Connections** 区域：
   - 找到要删除的连接
   - 点击 **"Delete"** 按钮

## 🗺️ 画布导航

### 平移画布
- 在空白区域拖动鼠标

### 缩放画布
- 使用 **鼠标滚轮** 缩放
- 点击 **"Reset Zoom"** 或按 **Ctrl+0** 重置为 100%

## 🎨 界面布局

界面分为三个主要区域：

```
┌─────────────────────────────────────────────────┐
│  工具栏 (按钮、统计信息)                          │
├─────────────────────────────────┬───────────────┤
│                                 │               │
│  节点编辑画布                    │  属性面板      │
│  (拖放、创建、连接)              │  (编辑详情)    │
│                                 │               │
└─────────────────────────────────┴───────────────┘
```

### 工具栏
- **Add Node**: 添加新节点
- **Delete Selected**: 删除选中的节点
- **Zoom**: 当前缩放比例
- **Reset Zoom**: 重置缩放
- **统计**: 显示节点和连接数量

### 属性面板
- **基本信息**: ID, Title, Content
- **位置**: X, Y 坐标
- **输入端口**: 列表和管理按钮
- **输出端口**: 列表和管理按钮
- **连接**: 与该节点相关的所有连接

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+N** | 添加新节点 |
| **Delete** | 删除选中的节点 |
| **Ctrl+0** | 重置缩放 |

## 💡 使用技巧

### 1. 快速创建工作流
- 双击多个位置快速创建多个节点
- 然后逐个连接它们

### 2. 组织节点
- 使用有意义的标题命名节点
- 通过内容添加描述或注释
- 合理布局节点位置

### 3. 端口管理
- 根据需要动态添加端口
- 使用描述性的端口名称
- 删除不需要的端口保持整洁

### 4. 调试连接
- 在属性面板查看节点的所有连接
- 确认输入输出是否正确连接

## 📖 示例工作流

### 创建一个简单的数据处理流程：

1. **创建输入节点**
   - 双击画布左侧创建节点
   - 重命名为 "数据输入"
   - 添加 1 个输出端口命名为 "原始数据"

2. **创建处理节点**
   - 双击画布中间创建节点
   - 重命名为 "数据处理"
   - 添加 1 个输入端口命名为 "输入数据"
   - 添加 2 个输出端口命名为 "结果1" 和 "结果2"

3. **创建输出节点**
   - 双击画布右侧创建两个节点
   - 分别重命名为 "输出A" 和 "输出B"
   - 各添加 1 个输入端口

4. **连接节点**
   - 从 "数据输入" 的 "原始数据" 连到 "数据处理" 的 "输入数据"
   - 从 "数据处理" 的 "结果1" 连到 "输出A"
   - 从 "数据处理" 的 "结果2" 连到 "输出B"

5. **完成！**
   - 你创建了一个简单的数据处理工作流

## 🐛 常见问题

### 双击创建了多个节点？
- 确保你是在 **空白区域** 双击
- 在节点上双击不会创建新节点

### 无法删除节点？
- 确保先 **选中** 节点
- "Delete Selected" 按钮应该变为可用状态

### 属性面板没有显示内容？
- 确保你 **选中了一个节点**
- 未选中时会显示 "Select a node to view properties"

### 连接创建失败？
- 确保从 **输出端口** 拖到 **输入端口**
- 检查两个端口是否属于不同的节点
