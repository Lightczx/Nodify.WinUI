# 测试指南

## 概述
本文档提供了完整的功能测试指南，帮助验证节点编辑器的所有功能是否正常工作。

## 已修复的问题

### 1. Zoom 属性错误
- **问题**: SamplePage 使用了不存在的 `Zoom` 属性
- **修复**: 改用 `ViewportScale` 属性
- **影响文件**:
  - `SamplePage.xaml` - 更新绑定
  - `SamplePage.xaml.cs` - 更新代码引用

### 2. DeleteSelectedNodeCommand 缺失
- **问题**: ViewModel 缺少删除选中节点的命令
- **修复**: 添加了 `DeleteSelectedNodeCommand` 和 `DeleteSelectedNode()` 方法
- **功能**: 支持通过工具栏和键盘快捷键删除选中节点

## 自动化测试建议

### 单元测试
```csharp
[TestClass]
public class NodeViewModelTests
{
    [TestMethod]
    public void AddInputPort_ShouldIncreasePortCount()
    {
        // Arrange
        var node = new NodeViewModel(new NodeModel("Test", 0, 0));
        var initialCount = node.InputPorts.Count;
        
        // Act
        node.AddInputPort("New Port");
        
        // Assert
        Assert.AreEqual(initialCount + 1, node.InputPorts.Count);
    }
}
```

### 集成测试
```csharp
[TestClass]
public class ConnectionTests
{
    [TestMethod]
    public void CreateConnection_ShouldUpdateBothPorts()
    {
        // Arrange
        var editor = new NodeEditorViewModel();
        var node1 = CreateTestNode(editor);
        var node2 = CreateTestNode(editor);
        
        // Act
        var connection = CreateConnection(
            node1.OutputPorts[0], 
            node2.InputPorts[0]
        );
        
        // Assert
        Assert.IsTrue(editor.Connections.Contains(connection));
    }
}
```
